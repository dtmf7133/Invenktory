您好，欢迎来到 Invenktory！

Invenktory 是一款轻量且简单的内部软件，用于跟踪您的设备。

Invenktory 根据 GPLv3 许可证发布，按原样提供，我们对其误用不承担任何责任。

第一步，使用左侧面板密码和盐字段创建要插入配置文件中的哈希值。请记住手动设置盐值。

当您要使用有限的 Web 服务器或 phpfpm 用户在 PHP 进程上下文中运行 Invenktory 时，您必须遵循一些简单的指令以获得最佳的首次设置：
  1. 检查网络应用专用路径中“Inventory”文件夹的权限；并在配置文件中设置其路径。
  2. 在清单路径中创建一个“.INVsampledir”文件夹并授予该文件夹写入权限。此文件夹将是从清单路径内的新文件夹复制的示例文件夹。
  3. 同样，Inventory 路径中必须存在“.INVsamplefile.xml”，并赋予该文件写入权限。此文件将是从新清单文件复制的示例文件。
  4. 根据需要配置每个 xml 文件的最大设备数（默认值：10）。
  5. 根据需要配置最大历史记录项和最大召回历史记录项（默认值：50）。

Invenktory 理解一组有限的命令和一组非常有限的参数：
cd、cd..、cp、编辑、帮助、ls、mv、pwd、显示

在编辑模式下按[CTRL]+[X]退出或按[CTRL]+[S]保存。
