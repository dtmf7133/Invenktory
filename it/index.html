Ciao e benvenuto in Inventory!

Inventory è un software on-premise leggero e semplice per tenere traccia dei tuoi dispositivi.

Inventory è rilasciato sotto licenza GPLv3, viene fornito AS-IS e non ci assumiamo alcuna responsabilità per il suo utilizzo improprio.

Primo passaggio, utilizzare la password del pannello laterale sinistro e i campi salt per creare l'hash da inserire nel file di configurazione. Ricordati di impostare manualmente lì anche il valore del sale.

Poiché eseguirai Inventory nel contesto del processo PHP, utilizzando un server Web limitato o un utente phpfpm, devi seguire alcune semplici direttive per una prima configurazione ottimale:
  1. Verifica i permessi della tua cartella "Inventario" nel percorso privato della tua app web; e imposta il suo percorso nel file di configurazione.
  2. Nel percorso dell'Inventario crea una cartella ".INVsampledir" e dai a questa cartella il permesso di scrittura. Questa cartella sarà la cartella di esempio da copiare dalle nuove cartelle all'interno del percorso dell'inventario.
  3. Allo stesso modo, nel percorso dell'inventario deve esistere ".INVsamplefile.xml" e dare a questo file il permesso di scrittura. Questo file sarà il file di esempio da copiare dai nuovi file di inventario.
  4. Configura il numero massimo di dispositivi per file xml come richiesto (impostazione predefinita: 10).
  5. Configurare gli elementi della cronologia massima e gli elementi della cronologia dei richiami massimi come richiesto (impostazione predefinita: 50).

Inventory comprende un insieme limitato di comandi con un insieme di parametri molto limitato:
cd, cd.., cp, modifica, guida, ls, mv, pwd, mostra

In modalità modifica premi [CTRL]+[X] per uscire o [CTRL]+[S] per salvare.
